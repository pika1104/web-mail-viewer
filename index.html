📧 Gmail連携設定

Gmail APIを使ってメールを取得します。Google Cloud Consoleで取得した情報を入力してください。

※ Gmailの「アプリパスワード」を使用します。
Google アカウント → セキュリティ → 2段階認証 → アプリパスワード で生成できます。
または Gmail API (OAuth) を使う場合は別途サーバーが必要です。
